FMDatabaseQueue Class Reference

Declared in FMDatabaseQueue.m

Overview

A key used to associate the FMDatabaseQueue object with the dispatch_queue_t it uses. This in turn is used for deadlock detection by seeing if inDatabase: is called on the queue’s dispatch queue, which should not happen and causes a deadlock.
